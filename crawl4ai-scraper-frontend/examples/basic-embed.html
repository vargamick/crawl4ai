This example demonstrates the simplest way to embed the Crawl4AI scraper interface into your web page using just a script tag and a single JavaScript call.

Integration Code:

<script src="../dist/crawl4ai-scraper.min.js"></script>
<script>
  Crawl4AIScraper.embed('#scraper-container', {
    apiUrl: 'http://localhost:8000'
  });
</script>
